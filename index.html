1~3교시 (3h): 비즈니스 데이터 리터러시

이론 & 사례

종합상사 업무 속 데이터 흐름(ERP-SAP-Excel)과 자동화에 최적화된 데이터 관리 원칙을 배웁니다.

  • 엑셀 지옥에서 벗어나는 데이터 전처리 원칙
  • 자동화가 가능한 데이터 vs 불가능한 데이터 구분법
  • 사전준비: 실무 엑셀 샘플 (민감정보 마스킹)

4교시 (1h): RPA vs AI Agent 마인드셋

마인드셋

자동화가 일자리를 위협하는 것이 아니라 커리어 생존력을 높이는 도구임을 이해합니다.

  • 단순 반복(RPA)을 넘어 판단(AI)하는 자동화의 가치
  • 자동화가 내 커리어에 주는 이점

5~7교시 (3h): n8n으로 만드는 나만의 AI 비서

실습 (입문)

n8n 인터페이스를 익히고, 뉴스 수집부터 메신저 전송까지 이어지는 워크플로우를 직접 설계합니다.

  • n8n 노드(Node) 기반 워크플로우 설계 입문
  • [실습] 글로벌 원자재/운임 정보 요약 보고서 자동화
  • 사전준비: n8n Desktop 설치, OpenAI API Key

1~2교시 (2h): PAD 기초와 웹 스크래핑

실습

Power Automate Desktop 환경을 설정하고 공시 정보를 엑셀로 자동 수집하는 기초를 다집니다.

  • 웹 접속 → 정보 추출 → 엑셀 기록 워크플로우
  • [실습] 관세청/환율 정보 자동 수집
  • 사전준비: PAD 설치, Chrome 확장프로그램

3~4교시 (2h): [실습 1] SAP-Excel 연동 자동화

실습 (심화)

엑셀 데이터를 기반으로 SAP에서 대량의 전표를 조회하고 결과를 기록하는 과정을 자동화합니다.

  • SAP GUI 스크립팅을 활용한 자동화 기초
  • 엑셀 송장번호 읽기 → SAP 조회 → 엑셀 업데이트
  • 사전준비: SAP GUI 스크립팅 활성화 (IT팀 협조 필수)

5~6교시 (2h): [실습 2] PO 등록 자동화 (OCR)

실습 (심화)

비정형 데이터인 PDF나 이메일에서 정보를 추출하여 SAP 특정 T-Code에 입력하는 실습입니다.

  • 메신저/PDF 수신 → 정보 추출 → SAP 입력
  • 사전준비: 실무 PDF/메일 양식 샘플

7교시 (1h): 일상의 자동화 & 워크숍 마무리

회고

개인의 성장을 돕는 일상 자동화 사례를 살펴보고, 실무 적용을 위한 로드맵을 작성합니다.

  • 팀별 자동화 아이디어 발굴 및 발표

주요 실습 Workflow 미리보기

실제 강의에서 구축하게 될 핵심 자동화 파이프라인입니다. Mitsui Korea의 실무에 즉시 적용 가능한 Power Automate 실습을 우선적으로 확인하세요.

🏢

SAP 대량 전표 조회 (PAD)

엑셀 리스트를 바탕으로 SAP를 자동 조작하여 전표 상태를 일괄 확인합니다.

📑

구매주문(PO) 등록 (PAD)

이메일 PDF의 데이터를 AI로 추출하여 SAP T-Code에 자동 입력합니다.

🌐

글로벌 마켓 인사이트 (n8n)

원자재/운임 뉴스를 AI가 분석하여 매일 아침 메신저로 보고합니다.

Workflow 1: PAD 기반 SAP 대량 전표 조회 및 검증

📄
Step 1: Read
관리 중인 엑셀 파일 송장 리스트업
🔄
Step 2: Loop
엑셀 각 행을 순차적으로 읽어 반복 실행
🖥️
Step 3: SAP Action
SAP GUI 조작 → 화면 캡처 및 데이터 추출
💾
Step 4: Update
조회 결과를 원본 엑셀에 다시 기록

Workflow 2: PAD 기반 구매주문(PO) 자동 등록

📩
Step 1: Receive
이메일 첨부 PDF 또는 폴더 내 파일 감지
🔍
Step 2: AI OCR
비정형 PDF에서 품명, 수량, 가격 추출
⌨️
Step 3: SAP Entry
추출 데이터를 SAP 전용 T-Code에 입력
✅
Step 4: Verify
입력 성공 여부 확인 및 담당자 알림

Workflow 3: n8n 기반 마켓 인사이트 에이전트

⏱️
Step 1: Trigger
매일 오전 8시 자동 실행 (Schedule)
🔍
Step 2: Collect
구글 뉴스 등 키워드 뉴스 수집 (HTTP)
🧠
Step 3: AI 요약
영업 관점 핵심 3문장 요약 (LLM)
💬
Step 4: Action
개인 카카오톡/슬랙 보고서 전송
